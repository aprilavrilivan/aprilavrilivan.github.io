Part B 2.3

  1. In this section, I implemented feature matching between two corresponding images using the extracted feature descriptors. First, I created a 500√ó500 matrix where each entry stores the Euclidean distance between two feature vectors (one from the left image and one from the right image). For each feature vector in the left image, I found the smallest distance d_1 and the second smallest d_2, then computed their ratio d_1 / d_2. If this ratio is smaller than 0.7, I considered the pair as a reliable match and kept it for homography matrix estimation. Finally, I visualized these matched pairs by drawing connecting lines between the two images, as shown below.

Part B 2.4

  1. In this section, I compute the final homography matrix using RANSAC. First, I randomly select four pairs of matched feature points to estimate a candidate homography matrix. Then, I project all feature points from the left image to the right image using this matrix and measure their reprojection errors. Points with errors smaller than a certain threshold (e.g., 3 pixels) are considered inliers. I repeat this process for many random samples (around 2000 iterations) and keep the homography that yields the largest number of inliers. Finally, I recompute the homography using all inlier correspondences judged by the best homography matrix to obtain a more accurate and robust result.
